## Project Background

**State of Mind** is an interactive geographic educational game collaboratively developed by our four-person team in the "Web Mapping" course at TU Wien.

Centered around the theme of "Democracy and Regimes", the game aims to guide players in exploring different political systems around the world through an engaging, educational approach. We transformed dry geopolitical data and geographic knowledge into a lively and interactive web experience, allowing users to deepen their understanding of the global landscape while playing.

## Core Gameplay & Game Modes

To cater to knowledge challenges across different dimensions, we designed four unique game modes on the web platform, comprehensively testing players' geographic reserves, historical knowledge, and logical reasoning skills:

### 1. Classic Mode
A traditional map quiz mode. Players interact directly with the map, using hints about regime characteristics or basic geographic information provided by the system to locate the corresponding countries by clicking on the map in real-time.

### 2. Figure Mode
This mode introduces highly influential historical and political figures from around the world. Players must use the clues about the historical figure presented on the screen to deduce and locate their corresponding country.

<ZoomableImage 
    src="/images/projects/state-of-mind/figure.jpg" 
    alt="Figure Mode Interface" 
    caption="Figure 2: Figure Mode interactive interface."
/>

### 3. Drag N Drop
A hardcore geographic shape challenge. The system provides the geometric outlines of specific countries, and players need to categorize them based on their overall characteristics and drag the countries into the corresponding regime columns.

<ZoomableImage 
    src="/images/projects/state-of-mind/drag-n-drop.jpg" 
    alt="Drag N Drop Mode Interface" 
    caption="Figure 3: Drag N Drop interactive interface."
/>

### 4. Maple (Geography Facts Game)
This is a progressive geographic fact reasoning game. The system provides attribute clues for a specific country one by one (e.g., population, main language, capital name). Players must synthesize these clues to guess the correct country name in the bottom-left input box, or click the correct location on the map.

## Global Regime Dashboard

In addition to the core gameplay, we developed a professional Statistics Dashboard for the platform (Figure 5). This dashboard aggregates the massive underlying geographic and political database of the game:

<ZoomableImage 
    src="/images/projects/state-of-mind/dashboard.jpg" 
    alt="State of Mind Global Regime Statistics Dashboard" 
    caption="Figure 5: National Data Statistics Dashboard."
/>

* **Comprehensive Data Analysis:** The system collects and analyzes data from **190** countries and regions worldwide.
* **Regime Classification Visualization:** Intuitively categorizes global countries and regions into four major types: Democracies, Authoritarian Regimes, Hybrid Regimes, and Monarchies.
* **Dynamic Global Regime Map:** Below the dashboard, we used Web Mapping technologies to render a colorful global regime distribution map. This allows players to examine the real-world geopolitical landscape from a macroscopic perspective outside of gameplay.

## Conclusion

State of Mind is a successful attempt to perfectly blend serious topics with interactive web technologies. As a member of the development team, this project not only honed my skills in complex state management, map API integration, and frontend UI/UX design, but also gave me a profound appreciation for the massive potential of Web GIS in education and science popularization.
