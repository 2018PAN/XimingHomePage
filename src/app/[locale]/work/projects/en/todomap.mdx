## Project Background

This is a group project for the "Mobile Cartography" course at TU Dresden.

In daily life, many activities have spatial attributes, such as grocery shopping, medical appointments, or visiting old friends. However, traditional To-Do apps often focus solely on time, while navigation apps focus only on locations. Therefore, we wanted to develop an application that brings the schedule directly onto the map, allowing users to intuitively see where they need to complete tasks today.

Our primary target audience is the **elderly**. As they age, seniors may face declining vision, memory loss, or discomfort with complex smartphone interfaces. By binding to-do items to map locations, the app can effectively help them build spatial memory and reduce the cognitive load of planning daily trips.

## App Interface

To avoid making it feel complicated for seniors, the app's hierarchy is extremely flat. The bottom navigation bar contains only four core pages: **Map**, **Tasks**, **Calendar**, and **Settings**.

## Key Features

Catering to the habits of elderly users, we incorporated many age-friendly (Accessibility) designs into the functional interactions:

* **Visual Location Selection:** When adding a new task, users don't need to type out complex, long addresses. They can simply drag a pin on the map to confirm the location.
* **Tutorial:** A detailed onboarding guide appears when the app is opened for the first time. If seniors forget how to use it over time, they can rewatch it anytime via "Help & Tutorial" in the Settings.
* **Customizable UI:** Users can freely adjust the global **font size** via a slider based on their visual needs. It also supports Night Mode and theme color switching.
* **Notifications & Multi-language:** The settings page includes a dedicated notification toggle. To accommodate users from different linguistic backgrounds, we support five languages: Chinese, English, German, Dutch, and Indonesian.
