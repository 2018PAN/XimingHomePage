# 王晰明 | 制图学硕士作品集

基于 **Next.js 15**, **Once UI** 和 **TypeScript** 构建的响应式个人作品集。

## ✨ 核心特性

- 🌍 **全站双语 (i18n)**：支持中英双语切换，包括动态时区翻译。
- 🔍 **智能筛选器**：支持多选的项目标签过滤系统。
- 📱 **响应式设计**：针对移动端和桌面端深度优化的左右分栏布局。
- 🌑 **主题切换**：内置深色/浅色模式切换。

## 🚀 技术栈

- **框架**: Next.js (App Router)
- **样式**: Once UI / CSS Modules
- **国际化**: next-intl
- **部署**: Vercel
